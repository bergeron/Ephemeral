Ephemeral lets you send temporary encrypted messages. After you create the message, you are given a URL that allows the recipient to read the message. The message is destroyed upon being read.

Messages are encrypted with a 128 bit symmetric AES cipher. The key is never persistently stored on the server. The key is saved only in the message's URL, not the database. An attacker with database access would see only encrypted data. Ephemeral uses SSL/TLS. The maximum message length is 16000 characters.

Written in Go: https://github.com/bergeron/Ephemeral
