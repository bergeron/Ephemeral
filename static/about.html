{{define "about"}} Ephemeral {{template "top" .}}
Ephemeral lets you send messages that are destroyed when read. After you create the message, we give you a URL that allows the recipient to read the message. Upon being read, the note is destroyed. Messages are encrypted with a symmetric AES cipher. The key is in the URL, and we throw away they key so it is never stored in our database. If the server was stolen, there would be no way to decrypt the messages in the database. Written in Go.
